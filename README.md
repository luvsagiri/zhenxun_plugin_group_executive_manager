# zhenxun_plugin_group_executive_manager
真寻群管插件，可设置代理群管
# usage：
    设置超过数量限制的隐形群管理员
    简易群管
    PS：
    艾特全体是没有用的，不要打扰别人啦
    ------------------------
    群主、超级管理员 可用：
    设置执行管理+@
    取消执行管理+@
    ------------------------
    群主、群管理、执行管理 可用
    禁言+@+小时数
    踢+@
    PS：
    解禁用 禁言+@+0
