# zhenxun_plugin_group_executive_manager
真寻群管插件，可设置代理群管
usage：
    简易群管
    PS：
    艾特全体是没有用的，不要打扰别人啦
    ------------------------
    群主、超级管理员 可用：
    设置管理权限+@
    添加管理权限+@
    取消管理权限+@
    删除管理权限+@
    ------------------------
    群主、群管理 可用
    禁言+@+小时数
    踢+@
    PS：
    解禁用 禁言+@+0
